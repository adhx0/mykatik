# mykatik

Welcome to my repository 🎉  

Project ini adalah aplikasi web sederhana untuk menghitung berbagai macam rumus matematika seperti luas dan keliling bangun datar.  

---

## 🚀 Cara Menggunakan
1. Klik link berikut untuk membuka aplikasi:  
   👉 [Kalkulating Web](https://link-projectmu.com)  
2. Pilih menu bangun datar yang ingin dihitung.  
3. Masukkan nilai pada kolom yang tersedia.  
4. Tekan tombol **Hitung** dan hasil akan langsung muncul.
